### Подготовка
- [ ] Уведомить РКН об обработке ПД ([тык](https://gkhnews.ru/1483478-inicziator-i-rkn/))
- [ ] В повестку 1 онлайн-ОСС обязательно включаются вопросы:
	- [ ] кто будет администратором (для будущих собраний)
	- [ ] как администратор в будущем будет принимать сообщения о проведении собраний и бюллетени собственников
	- [ ] о продолжительности голосования (7-60 дней)
- [ ] Наши вопросы:
	- [ ] наделить преда полномочиями на заключение ДУ
	- [ ] наделить совет полномочиями:
		- [ ] подписывать документы
		- [ ] принимать решение о ремонте ОДИ
		- [ ] расторгнуть дог. с охраной, нанять консьержа
		- [ ] аренда за размещение объектов на ОДИ 

- [ ] ГИС ЖКХ
	- [ ] Проверить электронный паспорт дома
		- [ ] Должно быть:
			- [x] полное заполнение данных о доме и помещениях собственников в нем с полным и и корректным указанием ссылок на кадастровые номера дома и помещений в нем ✅ 2025-11-09
			- [x] отсутствие в электронном паспорте помещений, относящихся к общему имуществу ✅ 2025-11-09
			- [x] наличие зарегистрированных прав собственности на помещения с актуальными строгими персональными данными собственников (ФИО + СНИЛС) ✅ 2025-11-09
			- [ ] неизменность сведений о составе помещений и их собственниках на период проведения ОСС (от размещения сообщения до формирования протокола)
		- [ ] Риски манипуляций
			- [ ] Удаление администратором привязки с Росреестром, изменение площадей
	- [ ] Если не администратор в ГИС ЖКХ, то получить эти права:
		- [ ] через ГЖИ или УК

### Уведомление
- [ ] Минимум за 10 дней до начала голосования размещаем уведомление о проведении ОСС:
	- [ ] в ГИС ЖКХ
	- [ ] в подъезде
		- [ ] фиксируем дату начала подготовки и проведения собрания:
			- [ ] акт о размещении уведомления + фото ([ТЫЦ](https://docs.domonline.ru/item.php?id=397&name=akt-o-razmeshchenii-soobshcheniya-ob-obshchem-sobranii-sobstvennikov))

### Голосование
- [ ] Электронно - ГИС ЖКХ
	- [ ] Каждую неделю ОИС присылает администратору "шахматку" - промежуточные результаты голосования (вт или ср, если не приходят, то идем в саппорт).
- [ ] На бланке
	- [ ] указать (еще чеклист [тут](https://docs.domonline.ru/download/chek-list-dlya-proverki-byulletenya-pismennogo-resheniya-sobstvennika.pdf)):
		- [ ] ФИО по документу
		- [ ] СНИЛС
		- [ ] номер квартиры
		- [ ] номер и дата гос. регистр-ции права собственности
		- [ ] доля в праве собственности
		- [ ] общая площадь помещения
		- [ ] если родитель голосует за ребенка:
			- [ ] серию/номер свид-ва о рождении
			- [ ] копию свид-ва о рождении
	- [ ] актом фиксируем факт передачи бланка администратору
	- [ ] сделать фото с голосующим (это уже моя мысль)

### Итоги
- [ ] Протокол формируется автоматически
- [ ] Сообщение с итогами ОСС  (не обязательно, но будет плюсом):
	- [ ] Акт о размещении + фото ([ТЫЦ](https://docs.domonline.ru/item.php?id=400&name=akt-o-razmeshchenii-soobshcheniya-s-itogami-obshchego-sobraniya-sobstvennikov))
- [ ] после передачи протокола в ГЖИ в течение 10 дней уведомить РКН о прекращении обработки ПД
